The dashboard

This is my dashboard. There are many like it, but this one is mine.
My dashboard is my best friend. It is my life. I must master it as I must master my life.

We use BaseCamp as a reference toolthroughout the course. It contains the entire course curriculum as well as a calendar with all of the key dates.

We use Treehouse as a tool to learn inour own time - Treehouse has a range of video courses, quizzes and code challenges we use to supplement the sessions.

We use Slack as a tool to communicate with each other. Slack is where we go t ask for help, speek to the mentors or organise meet-ups with other students.
